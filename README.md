
o projeto é uma agenda de uma barbearia para a organização e marcação dos procedimentos.
  primeiramente aparecerá um menu para o usuário escolher se ele deseja marcar um novo horário,mostrar os horários já
marcados,desmarcar um horário que ja foi marcado ou sair.
  após a escolha da marcação do horário, apareceria a escolha de qual profissional se deseja realizar o procedimento, a
escolha do procedimento com seus respectivos preços e por fim apareceria os horários disponíveis. todos eles seriam rece-
bidos pela entrada padrão ou se houvesse uma interface gráfica,por cliques.
  se escolher mostrar os horarios ja marcados, sera imprimido todos os horários marcados de todos os clientes diferentes
(armazenados em um banco de dados).
  se escolher desmarcar um horário, seria imprimido todos os horários para se retirar um do agendamento
  se escolher sair o menu se fecha.
